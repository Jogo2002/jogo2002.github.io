Traffic Stops with SQL

Author

Josh Goldhaber

Introduction

In this project I use three policing datasets from the Stanford Open Policing Project (Pierson et al. 2020): New Orleans Police Department (LA), Madison Police Department (WI), and Louisville Metro Police Department (KY). All data wrangling is conducted entirely in SQL. I focus on two central questions: (1) how search rates vary across race and gender, and (2) whether hit rates (the percent of searches that find contraband) differ across agencies. The goal is to compare policing practices across these jurisdictions and identify meaningful differences in how traffic stops escalate into searches and how often searches actually find contraband.

I first explore the structure of the database to understand what tables are available and how they are named. I then build a combined dataset that stacks three agencies into a single table and adds consistent indicators for searches and contraband. Finally, I summarize search and hit rates by race and gender and visualize patterns across cities.

Build Three-City Dataset (SQL written once)

Code 
query_three_city <- "
SELECT *
FROM (
    -- New Orleans (recode outcome)
    SELECT
        'New Orleans' AS city,
        subject_race,
        subject_sex,
        date,
        time,
        CASE WHEN outcome = 'citation' THEN 1 ELSE 0 END AS citation_issued,
        CASE WHEN outcome = 'warning' THEN 1 ELSE 0 END AS warning_issued
    FROM la_new_orleans_2020_04_01
    WHERE subject_race IS NOT NULL

    UNION ALL

    -- Madison
    SELECT
        'Madison' AS city,
        subject_race,
        subject_sex,
        date,
        time,
        citation_issued,
        warning_issued
    FROM wi_madison_2023_01_26
    WHERE subject_race IS NOT NULL

    UNION ALL

    -- Louisville
    SELECT
        'Louisville' AS city,
        subject_race,
        subject_sex,
        date,
        time,
        citation_issued,
        warning_issued
    FROM ky_louisville_2023_01_26
    WHERE subject_race IS NOT NULL
) AS t
"

Load the cleaned data into R

Code 
three_city <- dbGetQuery(con_traffic, query_three_city)

Summaries

Code 
race_city_summary <- three_city |>
  group_by(city, subject_race) |>
  summarise(
    stops = n(),
    citation_rate = mean(citation_issued),
    warning_rate = mean(warning_issued),
    .groups = "drop"
  )
